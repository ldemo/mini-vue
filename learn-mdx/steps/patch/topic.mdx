我们实现了 `createApp` 函数对 `根组件` 的渲染。
一个稍有规模的应用，业务也不是三两下就完成的。
想要通过一个组件完成业务的同时，还能保持着一定的 `可维护性`，
天方夜谭。那就必然会触及到 `组件树` 的概念。
前文也提及到 `单文件组件` 是需要经过转换后才能正常使用。接下来的目标代码会使用
`jsx` 来描述，不熟悉 `jsx` 也不影响， 因为都先会用 `h` 函数重写。
<br />
明确本文的目标。  
[第一步，定义 `Comp` 组件，若 `setup` 返回值是函数，则作为组件的 `render` 函数。](focus://demo.jsx#1:9)  
[第二步，渲染 `Comp` 组件。](focus://demo.jsx#15)
<CH.Code>
```jsx demo.jsx
const Comp = {
	setup() {
		const msg = "I'm comp"
		return () => (
			<div>{msg}</div>
		)
	}
}

createApp({
	render() {
		return (
			<div id="demo">
				<span>hello world</span>
				<Comp />
			</div>
		)
	}
}).mount('#root')

```
